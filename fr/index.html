🔒 SecretMessage

Chiffrez des messages directement dans votre navigateur. Aucun stockage serveur. Aucun suivi. Fonctionne avec WhatsApp, Telegram, Email et bien plus.
Solidité : -
Astuce : si vous avez joint une photo, le bloc chiffré peut devenir volumineux. L’envoyer comme document est l’option la plus sûre.
Remarque : si la clé est incorrecte ou si le bloc a été modifié, le déchiffrement échouera car AES-GCM vérifie l’intégrité.

📘 Guide 🔒 Politique de confidentialité 📰 Pourquoi utiliser SecretMessage ? 🌍 Langues : EN TR ES PT FR DE AR
❓ Comment ça marche ?
1️⃣ Écrivez votre message (et ajoutez éventuellement une photo)
2️⃣ Choisissez une clé ou utilisez 🔑 Générer
3️⃣ Cliquez sur Chiffrer pour créer le bloc chiffré
4️⃣ Envoyez le bloc, idéalement comme document s’il est volumineux
5️⃣ Le destinataire ouvre l’onglet Déchiffrer, colle le bloc, entre la même clé et lit le message
