❓ How does it work?
1️⃣ Write your message (optionally add a photo)
2️⃣ Choose a key or use 🔑 Generate
3️⃣ Click Encrypt to create the encrypted block
4️⃣ Send the block, ideally as a document if it is large
5️⃣ The receiver opens the Decrypt tab, pastes the block, enters the same key and reads the message
